## 動機發想
因為現代人太常喝飲料，以至於時常忘了喝水，然而喝水又很重要，因此我們決定做一個telegram bot 來提醒大家喝水

## 所需設備
- Raspberry pi 3 
- USB 轉 TTL 傳輸線 
- 電腦
